
# 关于页面开发：

### 对于一个新的项目：
+ 熟悉公共文件CSS, JS, 第三方插件。
+ 熟悉现有的HTML风格，CSS风格，脚本代码风格。
+ 熟悉页面文件引用，部署测试。
+ 熟悉后台的数据交换方式，接口定义。

### 对于新页面的开发：
1. 熟悉其他页面的样式风格。
2. 拆分当前页面到可从其他页面复用的组件。
3. 尽量是用已有的CSS来写，保证样式统一，减少工作量。
4. 尽量用最简单的JS编写逻辑，暂不考虑复用及维护。
5. 尽量和后台书面确定接口列表，传参，返回值。

### 对于空余时间：
+ 从页面中总结组件（HTML+CSS+JS)。
+ 优化JS编写逻辑。
+ 熟悉相关业务。
